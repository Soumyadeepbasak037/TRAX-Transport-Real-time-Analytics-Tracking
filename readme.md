
# TRAX – Transport Real-time Analytics & Tracking

A modern real-time bus transport tracking and analytics system designed to improve route planning, passenger experience, and driver-side operations.

**🚧 Not yet completed — actively under development**

## 🧩 Features

### ✅ Completed
- User authentication (JWT)
- Driver login and vehicle selection
- Real-time GPS updates from driver app
- Nearest stop detection using geospatial logic
- Single-hop route suggestion
- Basic analytics endpoints
- Admin Dashboard

### 🔄 In Progress
- Full route recommendation engine
- Passenger and Driver side UI improvements
- Advanced analytics
